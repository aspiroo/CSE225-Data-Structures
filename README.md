# CSE225 Data Structures

A comprehensive collection of fundamental data structures implementations in C++ for the CSE225 course. 

## Contents

This repository contains implementations of the following data structures:

- **[Stack](./Stack)** - Array-based stack implementation
- **[Stack List](./Stack%20List)** - Linked list-based stack implementation
- **[Queue](./Queue)** - Array-based queue implementation
- **[QueueList](./QueueList)** - Linked list-based queue implementation
- **[PriorityQueue](./PriorityQueue)** - Priority queue implementation
- **[BinaryTree](./BinaryTree)** - Binary tree data structure
- **[Graph](./Graph)** - Graph data structure and algorithms
- **[Dynamic Array](./Dynamic%20Array)** - Dynamic array implementation
- **[Complex Numbers](./Complex%20Numbers)** - Complex number operations

## Purpose

This repository serves as a practical reference for understanding and implementing fundamental data structures in C++. Each implementation includes:

- Core data structure implementation
- Essential operations (insert, delete, search, etc.)
- Example usage and test cases
